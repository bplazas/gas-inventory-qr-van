# Van Inventory & QR Code - Google Apps Script + AppSheet

Simple inventory system for a delivery van:
- Mobile charge/discharge via QR scanning (AppSheet)
- Real-time stock update (Apps Script trigger)
- Printable QR labels in Sheets

## Files
- `inventoryUpdater.gs`: Updates inventory qty & timestamp
- `onTransactionsChange.gs`: Entry point for change detection
- `.gitignore`: Keeps secrets out

Built with clasp + VS Code.
